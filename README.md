# 🚀 Customer Churn Analytics Dashboard

## 📊 Project Overview

This project is a **Customer Churn Analysis Dashboard** built using **Python & Plotly**.
It helps businesses understand customer behavior and identify factors contributing to churn.

---

## 🎯 Key Features

* 📌 KPI Metrics (Total Customers, Churn Rate)
* 📊 Churn Analysis by Contract Type
* 📈 Customer Tenure Distribution
* 💰 Monthly Charges vs Churn
* 🌐 Internet Service-based Churn Analysis
* 🌙 Dark Theme (Modern UI)
* 📱 Fully Responsive Dashboard

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* Plotly (Interactive Visualization)
* HTML (Auto-generated dashboard)

---

## 📂 Project Structure

```
📁 churn-dashboard
│-- churn_data.csv
│-- dashboard.py
│-- premium_dashboard.html
│-- README.md
```

---

## ▶️ How to Run

1. Install dependencies:

```
pip install pandas plotly
```

2. Run the script:

```
python dashboard.py
```

3. Dashboard will automatically open in your browser 🎉
